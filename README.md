YGO HUB

YGO HUB is a powerful Yu-Gi-Oh! (YGO) card database app that allows users to browse over 14,169 cards in one place. The app provides a fast and intuitive way to search, explore, and view detailed information about every card in the game.

Users can access complete card details including name, type (Monster, Spell, Trap), attribute, level, ATK/DEF stats, and full effect descriptions. The application is designed to deliver a smooth experience with clean architecture and optimized performance.
